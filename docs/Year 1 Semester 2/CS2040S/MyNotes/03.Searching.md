## Binary Search

Binary search isÂ an efficient algorithm for finding an item from a sorted list of items. It works by repeatedly dividing in half the portion of the list that could contain the item, until you've narrowed down the possible locations to just one.
### Using recursion

```java
public static int binarySearch(int[] array, int target, int low, int high) {
    if (low > high) {
        return -1;
    }
    int mid = (low + high) / 2;
    if (array[mid] == target) {
        return mid;
    }
    if (array[mid] > target) {
        return binarySearch(array, target, low, mid - 1);
    } else {
        return binarySearch(array, target, mid + 1, high);
    }
}
```

### Using iterative method

```java
public static int binarySearch(int[] array, int target) {
    int low = 0;
    int high = array.length - 1;
    while (low <= high) {
        int mid = (low + high) / 2;
        if (array[mid] == target) {
            return mid;
        } else if (array[mid] < target) {
            low = mid + 1;
        } else {
            high = mid - 1;
        }
    }
    return -1;
}
```

## Peak finding

**Problem:** Given a function $f(x)$ we want to find the maximum value of the function. 

A modified version of binary search can be used to solve this problem:

``` java
FindPeak(A,n)
	if A[n/2 + 1] > A[n/2] then
		FindPeak (A[n/2 + 1 . . n], n/2)
	else if A[n/2 - 1] > A[n/2] then
		FindPeak (A[1 . . n/2 - 1], n/2)
	else A[n/2] is a peak; return n/2
```

**Key Invariants**

1. There exists a peak in the range `[begin,end]`
2. Every peak in `[begin,end]` is a peak in `[1,n]`
